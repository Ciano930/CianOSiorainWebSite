The man behind the moustache.

Who am I? I'm a passionate creative and eager hobbyist! After finishing college I decided I wanted to pursue web design and development as it tapped into my creative side. I'm currently learning new systems like BootStrap and Laravel to better deliver dynamic web applications.

Why I do what I do. The main reason I started building web applications was a desire to provide a creative platform for people. During college I had been asked to build websites before however never felt I had the skills needed. After I finished I realised that the only reason I didn't have those skills was that I had never tried learning them! So now I work daily teaching myself new technologies so I can develop aesthetic and practical modern websites.

How I can help you! As a person first developer my primary thoughts are always on the user. Having worked as a teacher for a number of years I've seen many people struggle with complicated UI and bad UX. So to better understand these issues I go through numerous prototypes, testing colours; theme and layout, alongside navigation and transitions, to ensure that any site I work on is easy to use and visually attractive!
